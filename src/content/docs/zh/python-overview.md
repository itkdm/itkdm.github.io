Python 以 **易读、易写、生态大** 闻名：

- Web 框架：Django、Flask、FastAPI
- 数据/AI：NumPy、Pandas、PyTorch、TensorFlow
- 脚本/自动化：运维脚本、爬虫、办公自动化

与 Java 对比：
- 动态类型，上手快，开发效率高
- 性能不及编译型语言，但可用 C 扩展或多进程/协程提升
- 适合快速验证、原型或数据管线
