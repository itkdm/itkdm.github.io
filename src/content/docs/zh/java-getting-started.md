本篇聚焦“能跑起来”：

1. 安装 JDK（建议采用 Temurin 或官方 JDK 17/21 LTS）
2. 配置环境变量：`JAVA_HOME`、`PATH`
3. 编译与运行：
   - `javac Hello.java`
   - `java Hello`
4. 使用构建工具（可选）：快速创建 Maven / Gradle 项目骨架

小提示：

- 初学阶段 Maven 更常见，Gradle 更灵活。二者择一即可。
- IDE 推荐 IntelliJ IDEA 社区版即可满足入门需求。
