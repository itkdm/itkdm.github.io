本系列「入门」文档面向两类同学：

- **计算机及相关专业的初学者**：希望系统地打好基础，理解现代开发最常用的一些工具与工作方式。
- **非计算机专业、但希望在 AI 时代提升数字与信息素养的同学**：希望能看懂技术文档、能在 GitHub 等平台上自助学习与协作。

这一分类不会深入到某一种语言或框架的细节，而是聚焦在「横向通用」的基础技能——不论你未来是做前端、后端、数据、还是更多与 AI 相关的工作，这些能力都会长期受用。

### 你可以在这里学到什么？

- **如何阅读和书写 Markdown 文档**
  - 为什么几乎所有开源项目、技术博客、学习资料都在用 Markdown。
  - 最常用的语法：标题、列表、代码块、链接、图片等。
  - 写出「结构清晰、可维护」技术笔记的小技巧。

- **如何使用 Git 和 GitHub 做版本管理与协作**
  - 基本概念：仓库（Repository）、提交（Commit）、分支（Branch）、合并（Merge / Pull Request）。
  - 本地与远程的简单工作流：如何把本地项目推送到 GitHub，如何从 GitHub 拉取更新。
  - 作为初学者如何参与开源、如何给别人提 Issue / PR。

- **如何高效阅读技术文档与开源项目**
  - 快速判断一个仓库是否「活跃」「可靠」。
  - 顺着 README / 文档的推荐路径学习，而不是从零开始「盲看源代码」。
  - 学会利用搜索、目录、标签等信息架构，快速找到你要的内容。

- **在 AI 时代推荐补齐的基础技能**
  - 会用 AI（如 ChatGPT、Claude 等）来辅助学习、查错、生成示例代码。
  - 懂得基础的隐私与安全意识：不要在公有平台泄露敏感数据。
  - 知道如何结合「搜索引擎 + 官方文档 + AI 助手」来解决问题。

如果你是刚刚开始接触编程或技术世界，可以从这里按顺序阅读；如果你已经有一定基础，也可以把这里当作「技能清单」，查漏补缺或作为给朋友推荐的入门路径。
