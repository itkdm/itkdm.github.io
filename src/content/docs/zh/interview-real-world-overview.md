本系列聚焦 **真实 Java 面试题**，不是「网上拼题库」，而是：

- 来自真实的一线面试场景（校招 / 社招 / 内推），尽量保留原始提问方式与追问节奏。
- 每道题都会给出：
  - **题目原文**（包括面试官常见的追问、延伸问法）。
  - **考察点拆解**：到底在考 JVM、集合、并发、数据库，还是工程实践。
  - **示范回答思路**：分层次回答，先给主干，再按追问逐步展开。
- 适合在「复盘面试」「临近面试冲刺」时系统查漏补缺。

后续在这个小分类下会按主题继续扩展，例如：

- JVM 与内存模型相关的真实问答。
- 集合 / 并发 / 锁相关的深入追问。
- 数据库 / 事务 / 索引设计相关的问题。

你可以把本系列当作「面试复盘手册」：  
先从本篇概览了解整体结构，再针对薄弱环节阅读对应的题目解析。
