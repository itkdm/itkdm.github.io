本篇覆盖 Java 语言的必备语法与常用标准库：

## 核心语法
- 基本类型与引用类型
- `if / switch`、`for / while / enhanced for`
- 方法与参数、`var`（Java 10+ 局部推断）
- 面向对象：类、继承、多态、`interface`、`abstract`、`record`（Java 16+）

## 标准库速查
- 集合：`List` / `Set` / `Map` 及其常用实现（ArrayList、HashMap、LinkedHashMap）
- 工具类：`Objects`、`Collections`、`Arrays`
- 字符串处理：`String`、`StringBuilder`
- 时间日期：`java.time`（LocalDate/LocalDateTime/Duration/Period/Instant）

- 写一个用户注册表单的校验逻辑（必填、邮箱格式、长度）
- 写一个小型词频统计：读取文件 → 分词 → 计数 → 排序输出
