一个实用的 URL 解析工具，支持：

- **完整解析**：解析协议、主机名、端口、路径、查询字符串、哈希等
- **查询参数提取**：自动提取并表格展示所有查询参数
- **实时解析**：输入 URL 即时显示解析结果
- **一键复制**：点击任意字段快速复制
- **自动补全**：自动为不完整的 URL 添加协议前缀
- **错误提示**：无效 URL 时显示错误信息

适用于 URL 分析、API 调试、Web 开发等场景。
