一个实用的 URL 编码/解码工具，支持：

- **URL 编码**：将 URL 或文本转换为 URL 编码格式
- **URL 解码**：将 URL 编码字符串还原为原始文本
- **中文支持**：完美支持中文 URL 编码
- **实时编码**：输入即时显示编码结果
- **一键复制**：快速复制编码或解码结果

适用于处理 URL 参数、API 请求等场景。
