一个实用的 HTML 转义工具，支持：

- **HTML 转义**：将 HTML 特殊字符转换为实体字符
- **HTML 反转义**：将实体字符还原为原始字符
- **实时转换**：输入即时显示转义结果
- **安全处理**：正确处理 &、<、>、"、'、/ 等特殊字符
- **一键复制**：快速复制转义或反转义结果

适用于防止 XSS 攻击、HTML 代码处理等场景。
