一个便捷的 CSV 转 JSON 工具，支持：

- **CSV 解析**：正确解析 CSV 格式数据
- **自定义分隔符**：支持逗号、分号、制表符等分隔符
- **表头处理**：可选择第一行作为 JSON 对象的键名
- **格式化输出**：生成格式化的 JSON 数据，易于阅读
- **实时转换**：输入 CSV 即时显示 JSON 结果
- **选项配置**：支持去除首尾空格等选项

适用于数据处理、API 开发、数据迁移等场景。
