一个实用的 Markdown 预览工具，支持：

- **实时预览**：输入 Markdown 文本即时显示渲染效果
- **语法支持**：支持标题、列表、代码块、引用、表格等常见语法
- **代码高亮**：代码块清晰显示
- **左右分栏**：源码和预览并排显示，方便对比
- **快速清空**：一键清空内容重新开始

适用于 Markdown 文档编写、博客文章预览等场景。
