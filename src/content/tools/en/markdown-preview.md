A practical Markdown preview tool that supports:

- **Real-time Preview**: Instant rendering as you type Markdown text
- **Syntax Support**: Supports headings, lists, code blocks, quotes, tables, and more
- **Code Highlighting**: Clear display of code blocks
- **Split View**: Source code and preview side by side for easy comparison
- **Quick Clear**: Clear content with one click to start fresh

Perfect for writing Markdown documents, previewing blog posts, and more.
