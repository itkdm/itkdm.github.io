A convenient case conversion tool that supports:

- **Uppercase**: Convert all letters to uppercase
- **Lowercase**: Convert all letters to lowercase
- **Capitalize**: First letter of each line uppercase, rest lowercase
- **Title Case**: First letter of each word uppercase
- **Sentence Case**: First letter of each sentence uppercase
- **Toggle Case**: Reverse case of all letters
- **Multi-line Support**: Support multi-line text processing
- **One-click Copy**: Quickly copy conversion results

Perfect for text formatting, code standards, content editing, and more.
