A powerful regular expression testing tool that supports:

- **Real-time Testing**: Instant match results as you type regex
- **Multiple Flags**: Support global (g), case-insensitive (i), multiline (m), etc.
- **Highlight Display**: Matched items highlighted for easy viewing
- **Capture Groups**: Display all capture group contents
- **Preset Patterns**: Built-in common regex patterns (email, URL, phone, etc.)
- **Detailed Statistics**: Show match count, position, and more

Perfect for development debugging, text processing, and more.
