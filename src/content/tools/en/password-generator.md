A secure password generation tool that supports:

- **Customizable Length**: Adjustable from 4-128 characters
- **Character Type Selection**: Uppercase, lowercase, numbers, special characters
- **Exclude Similar Characters**: Optional exclusion of confusing characters (0, O, l, I, 1)
- **Password Strength Detection**: Real-time password strength level display
- **Batch Generation**: Generate multiple passwords at once (up to 20)
- **Cryptographically Secure**: Uses cryptographically secure random number generator

Perfect for account registration, password reset, and more.
