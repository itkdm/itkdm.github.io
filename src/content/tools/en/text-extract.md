A practical text extraction tool that supports:

- **Email Extraction**: Extract all email addresses
- **URL Extraction**: Extract HTTP/HTTPS links
- **Phone Extraction**: Extract phone numbers
- **Number Extraction**: Extract all numbers (including decimals and negatives)
- **IP Address Extraction**: Extract IPv4 addresses
- **Hashtag Extraction**: Extract #tag format tags
- **Multi-select Extraction**: Select multiple types for extraction simultaneously
- **Statistics**: Display extraction count for each type
- **Deduplication**: Automatically remove duplicates

Perfect for data mining, information extraction, content analysis, and more.
