A powerful text deduplication tool that supports:

- **Line Deduplication**: Remove duplicate lines
- **Retention Strategy**: Choose to keep first or last occurrence
- **Option Configuration**: Trim whitespace, case-sensitive, preserve order
- **Real-time Deduplication**: Instant deduplication results as you type
- **Statistics**: Display original line count, deduplicated count, removed count
- **One-click Copy**: Quickly copy deduplication results

Perfect for data processing, list organization, log analysis, and more.
