A convenient CSV to JSON conversion tool that supports:

- **CSV Parsing**: Correctly parse CSV format data
- **Custom Delimiters**: Support comma, semicolon, tab, and other delimiters
- **Header Handling**: Option to use first row as JSON object keys
- **Formatted Output**: Generate formatted JSON data for easy reading
- **Real-time Conversion**: Instant JSON results as you type CSV
- **Option Configuration**: Support options like trimming whitespace

Perfect for data processing, API development, data migration, and more.
