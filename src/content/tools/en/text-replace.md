A powerful text replacement tool that supports:

- **Plain Replacement**: Simple text find and replace
- **Regular Expressions**: Support regex pattern matching
- **Option Configuration**: Case-sensitive, replace all matches
- **Real-time Replacement**: Instant replacement results as you type
- **Statistics**: Display replacement count
- **Error Handling**: Display prompts for regex errors

Perfect for batch text processing, code refactoring, content editing, and more.
