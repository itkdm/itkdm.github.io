A pure frontend WeChat Moment screenshot generator tool with the following features:

- **Multiple Types**: Text only, shared webpage/article, single image, 9-grid images
- **Customizable Content**: Username, avatar, text content, location, forward source
- **Time Control**: Set post date and screenshot date/time
- **Interactive Elements**: Like count, like avatars, comment list
- **UI Styles**: Supports 7.0+ white interface, iOS-style notification bar, custom notification bar image
- **Emoticon Support**: Automatically recognizes and displays WeChat emoticons
