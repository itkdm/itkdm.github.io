A convenient image Base64 encoding tool that supports:

- **Image to Base64**: Upload images and automatically convert to Base64 encoding
- **Base64 to Image**: Paste Base64 encoding to restore images
- **Drag & Drop Upload**: Support drag-and-drop image file upload
- **Multiple Formats**: Support JPG, PNG, GIF, WebP and other common image formats
- **Image Preview**: Real-time image preview
- **File Information**: Display file name, size, type and other information
- **One-click Copy**: Quickly copy Base64 encoding

Perfect for frontend development, image embedding, API data transmission, and more.
