- **URL Encoding**: Convert URLs or text to URL-encoded format
- **URL Decoding**: Restore URL-encoded strings to original text
- **Chinese Support**: Perfect support for Chinese URL encoding
- **Real-time Encoding**: Instant encoding results as you type
- **One-click Copy**: Quickly copy encoded or decoded results

Perfect for handling URL parameters, API requests, and more.
