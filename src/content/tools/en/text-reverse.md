A flexible text reversal tool that supports:

- **Character Reverse**: Reverse character order of entire text
- **Line Order Reverse**: Keep line content unchanged, reverse line order
- **Both Reverse**: Reverse both characters and line order simultaneously
- **Word Reverse**: Reverse word order within each line
- **Real-time Reverse**: Instant reversal results as you type
- **Multi-line Support**: Support multi-line text processing
- **One-click Copy**: Quickly copy reversal results

Perfect for text processing, data conversion, content editing, and more.
