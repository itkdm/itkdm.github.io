A powerful text sorting tool that supports:

- **Multiple Sort Methods**: Alphabetical, reverse, by length, by numeric value
- **Option Configuration**: Case-sensitive, remove empty lines, trim whitespace
- **Real-time Sorting**: Instant sorting results as you type
- **Multi-line Support**: Support multi-line text, each line as an independent item
- **One-click Copy**: Quickly copy sorting results

Perfect for list organization, data sorting, content organization, and more.
