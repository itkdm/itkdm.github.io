This is an example app that only uses cloud disk links, suitable for large file sharing scenarios.

## Features

- ✅ Supports multiple cloud disk platforms
- ✅ Supports extraction code display
- ✅ Suitable for large file sharing

## Use Cases

Suitable for:
- Large files that are not suitable for direct download
- Long-term storage and sharing needs
- More stable access for domestic users

## Supported Cloud Disks

- Baidu Netdisk: Supports extraction codes
- Quark Netdisk: High-speed downloads
- Aliyun Drive: No speed limits
- Other cloud disks: Lanzou Cloud, 123 Cloud, etc.
